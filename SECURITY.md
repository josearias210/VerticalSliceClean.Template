# Security Policy

## Supported versions

Security updates are currently provided for the latest `main` branch state.

## Reporting a vulnerability

If you discover a security issue, please report it privately:

- Open a private security advisory on GitHub.

Please include:

- Affected component/file
- Reproduction steps
- Impact assessment
- Suggested fix (if available)

## Response targets

- Initial acknowledgment: within 72 hours
- Triage/update: within 7 days
- Remediation timeline: depends on severity and complexity

Please avoid posting vulnerability details in public issues until a fix is available.
